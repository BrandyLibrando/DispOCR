DispOCR - Electronic Display OCR Tool
=====================================
**An OCR solution for recognizing text and numbers from electronic displays, all through a dedicated graphical interface.** 

Allows *selection of region of interest* (ROI), as well as automatic *data logging* based on a configurable frequency. The project can also act as a *controller for control systems*, responding based on detected text or numerical data.

-------------------------------------

Created in fulfillment of thesis requirements. Intended system is Raspberry Pi 5, tested on Raspberry Pi 5 and Windows 11 environments. GUI program and components written in Python 3.12 and QML (Qt 6.9). Built and distributed using Briefcase (on PySide6 framework).

*GUI program, and integration with OCR models authored by Julius Alvin Librando.*
